# @sap/cloud-sdk-vdm-service-contract-service

This package contains the OData VDM for the [Service Contract Service](https://api.sap.com/api/API_SERVICE_CONTRACT_SRV) of SAP S/4HANA Cloud.
This service is part of the following communication scenarios: Service Contract OData Integration (SAP_COM_0558).
You can find additional documentation for this service on [help.sap.com](https://help.sap.com:00443/http.svc/ahp2/SAP_S4HANA_CLOUD/latest/EN/40/e8f572cd8a4a41bfe34b3094ef638e/frameset.htm).
