# @sap/cloud-sdk-vdm-just-in-time-call-process-service

This package contains the OData VDM for the [Just In Time Call Process Service](https://api.sap.com/api/API_JIT_CALL_PROCESS_SRV) of SAP S/4HANA Cloud.
This service is part of the following communication scenarios: Logistics - Just In Time Inbound Integration (SAP_COM_0168).
You can find additional documentation for this service on [help.sap.com](https://help.sap.com:00443/http.svc/ahp2/SAP_S4HANA_CLOUD/latest/EN/e6/26b94c627e4f30b5b64b657dafbee4/frameset.htm).
